# AuroraFlow Journal

AuroraFlow Journal is a calming, privacy‑focused journaling prototype built with React Native and Expo.  
The goal of this project is to create a simple, emotionally supportive space where users can write, save, and revisit their thoughts without distractions.

## Features

- Aurora Borialis inspired gradient UI for a calming experience  
- Write journal entries in a clean, minimal interface  
- Save entries locally using AsyncStorage  
- View a scrollable list of past entries with timestamps  
- Keyboard‑aware layout for smooth typing on mobile  
- Persistent storage — entries remain even after closing the app  

## Tech Stack

- React Native  
- Expo  
- AsyncStorage  
- JavaScript
